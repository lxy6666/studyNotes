#### 编程题

###### 1.概述脚手架的实现过程

脚手架工具实为node.js的cli应用

1. package.js中添加bin字段，指定cli应用的入口文件
2. 创建cli.js 并指定文件头
3. 在文件中通过命令行交互 询问用户问题
4. 获取文件模板目录
5. 读取文件模板 使模板中的变量通过用户回答来赋值
6. 通过模板引擎渲染文件并写入到目标路径
